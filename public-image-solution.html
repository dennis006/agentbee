BEELLGROUNDS Welcome Image

Für Discord Bot Willkommensnachrichten

Anleitung:

  1. Erstellen Sie ein GitHub Repository
  2. Laden Sie Ihr BEELLGROUNDS-Bild hoch
  3. Klicken Sie auf das Bild → "Raw" Button
  4. Kopieren Sie die URL
  5. Verwenden Sie diese URL im Discord Bot

Beispiel URL: https://raw.githubusercontent.com/username/repo/main/beellgrounds-welcome.png
